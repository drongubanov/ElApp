# ElApp

Веб-приложение для расчёта мощности и тока в сетях постоянного тока,
однофазных и трёхфазных сетях переменного тока. Работает полностью в
браузере, без сервера и без сторонних зависимостей.

## Возможности

- Расчёт активной (P), полной (S) и реактивной (Q) мощности, тока и
  напряжения для DC, однофазной и трёхфазной (симметричной) сети.
- Ввод любой из двух связанных величин (мощность или ток) — остальные
  вычисляются автоматически.
- Подбор рекомендуемого номинала автоматического выключателя по
  расчётному току.
- Подбор рекомендуемого сечения кабеля (медь/алюминий) по справочной
  таблице допустимых токовых нагрузок (приближена к ПУЭ гл. 1.3) с учётом
  способа прокладки (воздух / труба или кабель-канал / лоток-пучок) и
  числа кабелей, проложенных рядом.
- Расчёт потери напряжения на линии по длине кабеля и сечению
  рекомендованной жилы, со сравнением с общепринятой нормой (≤5%).
- История расчётов (хранится локально в `localStorage`), с возможностью
  повторно открыть или удалить запись.
- **Конструктор сети** — построение электрической схемы в виде дерева
  узлов: графический редактор с кликабельными блоками, соединёнными
  линиями (кабелями), в духе диаграммы иерархии проекта. Корневой узел —
  точка ввода (главный щит), у него могут быть дочерние узлы — следующие
  щиты или потребители, и так до любой глубины. Узел может одновременно
  иметь собственную нагрузку (тип сети, напряжение, cosφ, мощность или
  ток) и дочерние узлы — например, щит, часть нагрузки которого
  подключена напрямую, а часть — через под-щиты. На самом блоке есть
  кнопки ⚙ (открыть панель свойств), + (добавить дочерний узел) и −
  (удалить узел, с подтверждением, если у него есть дочерние узлы) — они
  работают независимо от того, какой узел выбран сейчас. В панели
  свойств задаются параметры узла и линии, питающей его от родителя
  (способ прокладки, число кабелей рядом, длина), а также коэффициент
  одновременности (Кс) для суммирования нагрузки его дочерних узлов. По
  кнопке
  «Рассчитать сеть» для каждого узла дерева — от листьев к корню —
  считается суммарная нагрузка, подбираются автоматический выключатель и
  кабель линии, считается падение напряжения на ней; результат и
  рекомендуемый автомат отображаются прямо на блоке и подробно — в панели
  свойств. Ошибка в одном узле блокирует расчёт только его самого и его
  предков, не затрагивая соседние ветви дерева. Схема сети сохраняется
  локально в `localStorage` и восстанавливается при следующем открытии
  страницы.

> Подбор автоматического выключателя и сечения кабеля носит справочный,
> ориентировочный характер. Расчёт ориентирован на действующую редакцию
> ПУЭ (гл. 1.3, 7-е издание) — таблицы 6-го издания устарели и для
> текущего проектирования не применяются, поэтому отдельное сравнение
> редакций в калькуляторе не приводится. Поправочные коэффициенты на
> способ прокладки и число кабелей — приближённые. Для проектирования
> реальных электроустановок используйте актуальные таблицы ПУЭ/ГОСТ с
> учётом температуры окружающей среды и реальных условий прокладки, либо
> обратитесь к специалисту.

## Запуск

Приложение — статический сайт без сборки. Достаточно открыть
`index.html` в браузере или поднять любой статический HTTP-сервер,
например:

```bash
npx serve .
# или
python3 -m http.server 8000
```

и открыть `http://localhost:<порт>/`.

## Используемые формулы

- **Постоянный ток:** `P = U · I`
- **Однофазная сеть переменного тока:** `S = U·I`, `P = U·I·cosφ`, `Q = √(S²−P²)`
- **Трёхфазная сеть (симметричная нагрузка):** `S = √3·U·I`, `P = √3·U·I·cosφ`, `Q = √(S²−P²)`,
  где `U` и `I` — линейное напряжение и линейный ток.
- **Потеря напряжения на линии:** `ΔU = k·ρ·L·I·cosφ/S`, где `k = 2` для DC и однофазной сети,
  `k = √3` для трёхфазной; `ρ` — удельное сопротивление жилы (медь 0.0175, алюминий 0.028 Ом·мм²/м);
  индуктивное сопротивление не учитывается (приближение для предварительной оценки).
- **Конструктор сети (узел дерева):** активная и реактивная мощности узла
  складываются из его собственной нагрузки и суммы P/Q всех его дочерних узлов
  (уже рассчитанных рекурсивно, от листьев к корню), с учётом коэффициента
  одновременности (`P = Кс·(Pсобств + ΣPдетей)`, аналогично для Q), затем
  `S = √(P²+Q²)` и ток узла считается по тем же формулам, что и для одного
  блока. Это упрощение (фазы и напряжения дочерних узлов не приводятся к
  общему вектору) — подходит для предварительной оценки схемы, а не для
  итогового проекта.

## Структура проекта

```
index.html           разметка приложения
css/styles.css       стили
js/calculations.js   формулы расчёта мощности и тока (без UI)
js/tables.js         справочные таблицы кабелей/автоматов и логика подбора
js/network.js        расчёт дерева узлов сети (конструктор сети)
js/history.js        хранение истории расчётов в localStorage
js/networkStorage.js хранение схемы сети конструктора в localStorage
js/format.js         форматирование чисел и единиц измерения
js/app.js            связывание UI с расчётной логикой
tests/               модульные тесты (Node.js test runner)
```

## Тесты

Модульные тесты для расчётной логики (`js/calculations.js`,
`js/tables.js`) на встроенном тест-раннере Node.js — внешние зависимости
не требуются:

```bash
npm test
```
